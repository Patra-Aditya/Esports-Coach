# 🎮 Esports Coach

An AI-powered application designed to provide competitive gaming insights and strategy analysis. This project utilizes a **Retrieval-Augmented Generation (RAG)** pipeline to offer personalized coaching based on specific gaming data.

---

## 🚀 Features
* **AI Strategy Analysis:** Leverages LLMs to analyze gameplay and provide tactical feedback.
* **Knowledge Retrieval:** Uses a RAG pipeline to pull relevant information from a custom gaming database.
* **Seamless Integration:** Built with a Django backend for a robust and scalable web experience.

## 🛠️ Tech Stack
* **Framework:** Django (Python)
* **AI Orchestration:** LangChain
* **LLM:** Google Gemini API
* **Vector Database:** ChromaDB
* **Frontend:** Django Templates & HTML/CSS

## 📋 Prerequisites
Before running the project, ensure you have:
* Python 3.10+
* A Google API Key
